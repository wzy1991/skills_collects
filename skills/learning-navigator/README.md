# Learning Navigator 使用说明

## 这个 Skill 做什么？

Learning Navigator 是一个“学习路径生成器”，用于帮助你在任何领域快速建立系统认知，并输出可执行的学习路线。

适用于：

- 行业分析（如：AI、半导体、新能源汽车）
- 学科学习（如：线性代数、微观经济学）
- 技能提升（如：Python、产品经理、写作）

---

## 使用方式

### 输入示例

- 我想 4 周入门 Python 数据分析
- 我想了解半导体行业
- 帮我制定线性代数学习路线

---

## 输出结构

系统会生成 5 大模块：

### 1. 领域全景地图
理解整个领域的结构

### 2. 80/20 核心概念
抓住最重要的 5~10 个核心知识

### 3. 灯塔资源
推荐经典书籍/报告/人物/网站

### 4. 四周学习路线
Week1~Week4 实战路径

### 5. AI 伴学 Prompt
用于持续训练和复习

---

## 学习方式建议

- 不要只读，要做输出
- 每周必须有成果
- 用费曼法讲给别人听
- 每周做一个小项目（MVP）

---

## 示例输出

如果输入：

> 我想学习 Python 数据分析

系统会输出：

- 数据分析知识地图
- Pandas / NumPy 核心概念
- Kaggle 数据集练习
- 4 周项目路线
